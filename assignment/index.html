Demolition Overview

Philadelphia hase faced with more and more building demolitions these years. This project aims to show all the demolitions by year.

This application shows each dot color coded based on years of demolition start time. To know more information about the building, click on that dot on the map.

After you click the dot, those dots with similar demolition years will also show on the map.
